Welcome to **My-DeepLearning**, a hands-on collection of Jupyter Notebooks designed to guide you through the fundamental building blocks and advanced architectures of deep learning. ---

## üìÇ Notebooks & Topics
- [**Activations_Functions.ipynb**](./Activations_Functions.ipynb) ‚Üí Demonstrates different activation functions (Sigmoid, ReLU, Tanh, etc.) and their behavior in neural networks.
- [**Cost_Loss_Functions.ipynb**](./Cost_Loss_Functions.ipynb) ‚Üí Explains and visualizes different cost/loss functions (MAE, MSE, RMSE, Hinge, Binary Cross-Entropy, Categorical Cross-Entropy).
- [**Word_Classification.ipynb**](./Word_Classification.ipynb) ‚Üí Performs word/text classification using deep learning models with TensorFlow/Keras.
- [**Gradient_Descent/**](./Gradient_Descent) ‚Üí Fundamentals of Optimization: A conceptual breakdown of the Gradient Descent algorithm, visualizing how models learn by minimizing the cost function.
- [**sgd_vs_gd/**](./sgd_vs_gd) ‚Üí Optimization Comparison: Illustrates the differences and practical trade-offs between Stochastic Gradient Descent (SGD) and Batch Gradient Descent (GD).
- [**churn_prediction/**](./churn_prediction) ‚Üí Industry Application: A real-world project demonstrating how to build a deep learning model for customer churn prediction.
- [**Precision_recall.ipynb**](./Precision_recall.ipynb) ‚Üí Evaluation Metrics: A focused notebook explaining and visualizing the key classification metrics: Precision, Recall, and the F1-Score.
- [**Dropout layer/**](./Dropout%20layer) ‚Üí Regularization Technique: Explores the implementation and impact of Dropout, a vital technique for preventing overfitting in large neural networks.
